# 🎮 Simon Says Game

An interactive memory-based game built using HTML, CSS, and JavaScript where players must repeat an increasingly complex sequence of colors.

---

## 🚀 Features

- 🎯 Random sequence generation
- 📈 Level progression system
- ✅ User input validation
- 💡 Visual flash animations
- 🔊 Sound effects for each button
- 🔇 Mute / Unmute functionality
- 🏆 Score and high score tracking
- 🎮 Start button for better UX

---

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript (Vanilla JS)

---

## 🎯 How to Play

1. Click on **Start Game**
2. Watch the sequence carefully
3. Repeat the same sequence by clicking buttons
4. Each correct round increases difficulty
5. One mistake = Game Over
